# Касса для ресторана (POS)

Кассовый аппарат для ресторана: столики, заказы из меню, подсчёт суммы и
количества, печать чека. Самодостаточное PWA-приложение (отдельно от трекера
тренировок в корне репозитория).

Live: https://dimahodonovich22.github.io/gym-tracker-test/restaurant/

Стартовое меню предзагружено из меню ресторана **Vyshnia House** (134 позиции:
72 блюда + напитки, цены в €). Любую позицию можно отредактировать на экране
«Меню». Меню версионируется (`menuVersion`) — при обновлении канонического списка
оно подтягивается автоматически.

## Возможности
- **Столики** — добавление, статус «занят/свободен», текущая сумма.
- **Заказы** — блюда из меню с количеством (+/−), подсчёт суммы и количества,
  опциональный процент обслуживания.
- **Меню** — добавление/изменение/удаление блюд по категориям.
- **История** — закрытые (оплаченные) чеки, итог выручки, повторная печать.
- **Чек** — свёрстан под чековый термопринтер (лента ~80 мм), печать через диалог
  печати браузера.
- **Настройки** — название заведения, валюта (по умолчанию €), % обслуживания,
  адрес/телефон для шапки чека.

## Установка на планшет
1. Открыть `https://dimahodonovich22.github.io/gym-tracker-test/restaurant/` в
   браузере планшета.
2. «На экран „Домой"» → касса ставится как отдельное приложение «Касса» и
   открывается на весь экран. Работает офлайн (service worker).

## Технически
- Файлы: `index.html`, `app.js`, `styles.css`, `icons.js`, `sw.js`,
  `manifest.json`, `icon-192.png`, `icon-512.png` — всё внутри этой папки.
- Хранение: `localStorage`, ключ `restpos.test.v1` (независимо от трекера).
- Кэш service worker: `restpos-test-v1`.
- Печать: `@media print` рендерит только чек шириной 80 мм, остальной интерфейс
  скрывается.
