# Каса — Vyshnia House

Касовий застосунок для ресторану: столики, замовлення з меню, підрахунок суми
й кількості, друк чека. Самодостатній PWA (працює офлайн, ставиться на iPad/
телефон як застосунок).

## Можливості
- **Столики** — додавання, статус «зайнятий/вільний», поточна сума.
- **Замовлення** — страви з меню з кількістю (+/−), сума, кількість, ПДВ (A/C).
- **Меню** — додавання/зміна/видалення страв і цін за категоріями.
- **Чек** — у форматі бельгійського BTW-кас-чека (80 мм), з розбивкою BTW і
  блоком Controlegegevens.
- **Історія** — закриті чеки, згруповані по днях; денний Z-звіт.
- **Друк:**
  - **Epson ePOS-Print** — прямий друк по Wi-Fi на мережевий принтер
    (напр. TM-m30III): Налаштування → «Принтер (Wi-Fi)» → IP → «Тест друку».
  - **Браузерний друк** — запасний варіант (PDF / звичайний принтер).
- **Резервна копія** — експорт/імпорт усіх даних у файл `.json`.

## Налаштування принтера (Epson TM-m30III, Wi-Fi)
1. Підключити принтер до тієї ж Wi-Fi, що й iPad; дізнатися його IP.
2. У Safari відкрити `https://[IP принтера]` і довірити сертифікат (один раз).
3. У касі: Налаштування → «Принтер (Wi-Fi)» → ввести IP → «Тест друку».

## Технічно
- Файли: `index.html`, `app.js`, `styles.css`, `icons.js`, `sw.js`,
  `manifest.json`, `icon-192.png`, `icon-512.png`.
- Дані: `localStorage` (ключ `restpos.test.v1`).
- Друк ePOS-Print: ePOS-Print XML → `https://[IP]/cgi-bin/epos/service.cgi`.
- Чек на друк: `@media print` рендерить лише чек шириною 80 мм.
